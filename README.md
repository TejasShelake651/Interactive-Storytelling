# Interactive Storytelling

Hey! This is a cool interactive storytelling project. It's a web app where you dive into epic stories with kingdoms, warriors, magic, jungles and more. Built with HTML, CSS, and JavaScript - super simple, just open in your browser.

## Quick Start
1. Open `index.html` in any browser (Chrome, Firefox, etc.)
2. Start exploring the story!

## Technologies Used

- **HTML**: Structure and content of the web pages
- **CSS**: Styling and layout
- **JavaScript**: Interactive features and logic

## Pages
- **Home** (`index.html`): Main story interface with images like story.jpg, kingdom.png, warrior.png.
- **About** (`about.html`): More info on the project.

## Features
- Interactive elements powered by `script.js` and `about.js`.
- Cool styling in `index.css`.
- Themed images: jungle, magic, moon, etc.

Wanna contribute? Just edit the files and refresh!

Made for fun storytelling adventures.
